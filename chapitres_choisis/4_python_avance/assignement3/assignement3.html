1. Basics

1.1. Factorial

  • Write a generator that outputs the first n factorials values: 

    list( facGen( 5 ) )
  #=> [1,2,6,24,120]

1.2. Fizzbuzz

  • Solve the Fizz buzz problem using a python generator

2. Pipelines

2.1. Moving average

  • Implement a generator that computes the moving average of a number generator. The size of the window is passed on argument.

2.2. Prime numbers

  • Implement the Sieve of Erathostenes using a pipeline of generators
  • The trick is to add and chain generators dynamically during the evaluation

3. Tree traversals

3.1. Data structure

  • Write a small datastructures representing binary trees with values in the nodes

3.2. Traversals

  • Write two generators that traverses any tree in Depth-First and Breadth-First order.
